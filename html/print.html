Olá, Mundo!

Todo programa em rust necessita de uma função main:

  • fn main ( ) { } = esta linha define a função principal do programa;
  • print!(" ") = esta linha imprime a string no console;
  • ; são usados para finalizar as instruções em Rust.

Para imprimir você precisa passar um comando:

  • print!("Escreva aqui seu texto!");

Esse projeto muito mais do que ensinar códigos, busca te ensinar a resolver problemas, logo ao final de cada explicação, você irá encontrar a seguinte estrutura:

  • {00} Problema > Ao acessa esse link, será direcionado para a página do "Rust - Playground". Aqui vai encontrar o problema que precisa resolver.
  • {00} Solução > Ao acessar esse link, você irá encontrar uma ou mais soluções possíveis para o problema.
