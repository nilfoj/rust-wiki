{O3} Tipos Númericos!

Números Inteiros

  • u8, u16, u32, u64,u128: Números inteiros com sinal positivo.
  • i8, i16, i32, i64, i128: Números inteiros positivos e negativos;.
  • u:
    • u8: Armazena valores de 0 a 255 (1 byte);
    • u16: Armazena valores de 0 a 65535 (2 bytes);
    • u32: Armazena valores de 0 a 4.294.967.295 (4 bytes);
    • u64: Armazena valores de 0 a 18.446.744.073.709.551.615 (8 bytes);
    • u128: Armazena valores de 0 até 170.141.183.460.469.231.731.687.303.715.884.105.727 (16 bytes).
  • i:
    • i8: armazena qualquer valor no intervalo: -128 a 127;
    • i16: armazena qualquer valor no intervalo: -32.768 a 32.767;
    • i32: armazena qualquer valor no intervalo: 2.147.483.648 a 2.147.483.647;
    • i64: armazena qualquer valor no intervalo: -9.223.372.036.854.775.808 a 9.223.372.036.854.775.807;
    • i128: armazena qualquer valor no intervalo: -170.141.183.460.743.176.821.145.600 a 170.141.183.460.743.176.821.145.599;
