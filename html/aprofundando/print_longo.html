Hello, world!

Uma das primeiras coisas que fazemos tradicionalmente ao inicial o estudo em qualquer linguagem é imprimir na tela a frase: Hello, world! ou Olá, Mundo!

Essa tradição se deu por conta de Brian Kernighan, autor e criador do "C Programming Language". caso queira conhecer melhor, acesse os links a seguir:

  • Você conhece a origem do “Hello World!”?
  • “Hello World” em 30 Linguagens de Programação Diferentes

Cada linguagem possui sua própria forma de escrita e o rust não é muito diferente disso:

            
        
fn main () {

    println!("Hello World!");

}

Estrutura do código:

  • fn main () {} = Função principal do seu código (Aqui você deve escrever seu código principal entre os { });
  • println!("") = Imprimi um texto que está entre aspas.(Dentro das aspas você escreve o código que deseja que apareça na tela).

Porém, a ideia aqui é entender a importancia e uso do "print"

Tipos de print:

  • print! = imprimi um texto, mas não adiciona uma linha;
  • println! = imprimi um texto e pula uma linha;

Acesse: Rust Playground e faça os testes.

O print é muito importante na programação, por vezes nos deparamos com a necessidade de exibir um determinado código na tela, como: a saída de um IP, o resultado de um calculo, entre outras funcionalidades. Vou deixar a seguir alguns códigos aonde podemos observar seu uso.

                
            
// Soma

let x = 12;
let y = 7;

let resultado = x + y;

println!("Soma de x e y: {}", resultado);



// Informações de um pessoa
                        
let nome = "João";
let idade = 30;

println!("Nome: {} | Idade: {}", nome, idade);
